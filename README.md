# Dennys Heras - Portfolio Website

A modern, responsive portfolio website built with React + Vite for Dennys Heras, a telecommunications technician transitioning into software development.

## Features

- **React 18** with **Vite** for fast development and optimized builds
- **Zero external CSS libraries** - Pure CSS with custom properties (variables)
- **Mobile-first responsive design** - Works perfectly on all devices
- **Dark theme** - Purple & blue accent colors
- **Inter font** from Google Fonts
- **Modular component architecture** - Easy to maintain and extend
- **Smooth animations** - CSS-only transitions, no heavy libraries
- **SEO optimized** - Proper meta tags and semantic HTML
- **Scalable projects data** - Easy to add new projects

## Getting Started

### Prerequisites

- Node.js 16+ and npm/yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build locally
npm run preview
```

The development server will open at `http://localhost:5173`

## Customization

### Adding/Editing Content

All text content is centralized in `src/data/content.js`:

```javascript
export const content = {
  hero: { /* ... */ },
  about: { /* ... */ },
  // ... etc
}
```

### Adding Projects

Edit `src/data/projects.js` to add new projects:

```javascript
export const projects = [
  {
    id: 'my-project',
    title: 'My Project Title',
    status: 'production',
    description: '...',
    stack: ['Tech1', 'Tech2'],
    repoUrl: 'https://...',
    metrics: [...],
    flow: [...]
  }
]
```

### Design System

All colors, spacing, and typography are defined as CSS custom properties in `src/App.css`:

```css
:root {
  --bg-primary: #0d0d0f;
  --accent-purple: #7c3aed;
  --accent-blue: #2563eb;
  /* ... more variables */
}
```

Edit these variables to change the entire site's appearance.
